# Sponsor

CPA Orbit is independently maintained. Sponsorship supports maintenance, tests, documentation, and cross-platform releases; it does not buy feature priority or weaken the project's privacy boundaries.

## What support funds

- Reproducing and fixing real operating issues.
- Windows and macOS release work.
- Regression tests and security review.
- English and Chinese documentation.
- Release, signing, and infrastructure costs.
