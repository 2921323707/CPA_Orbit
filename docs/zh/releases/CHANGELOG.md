# 更新日志

## 待发布 · v1.3.0

- 新增 Sub2API 主号池、官方 Codex session 导入、CPA 部署兜底、主备绑定、迁移回滚和单一活动主绑定保护。
- 新增号池运维台：网关健康、活动绑定、操作记录、手动采集、7 天 Token 曲线和陈旧快照提示。
- 新增本地 SQLite 控制面，按 15 分钟记录请求、Token、延迟与成本聚合，最长保留 90 天。
- 导入可一次完成“归档 + 部署”；运行池失败不会丢弃本地归档。
- 删除订阅前先安全撤销 Orbit 托管账号；外部接管账号只解绑、不误删。
- Sub2API 管理密钥只写不回显，远程目标必须显式允许并使用 HTTPS。

## v1.2.1

- 新增原生 macOS Apple Silicon CI 构建，并发布 ARM64 DMG、ZIP 和 SHA-256 校验文件。
- 在 macOS 打包流程中加入 Mach-O 架构检查和 DMG 验证。
- 修正 Wails 生成的 `CPA Orbit.app` 应用包路径。
- 感谢 [@tomatokoko](https://github.com/tomatokoko) 在 [issue #9](https://github.com/2921323707/CPA_Orbit/issues/9) 中提出原生 Apple Silicon 支持需求并推动本次发布。

## v1.1.0

- 文档默认语言改为英文，并加入对应路由的简体中文切换。
- 统一固定左侧导航层级，强化当前页面高亮。
- 重排项目简介，加入紧凑摘要和本地运行动态反应堆。
- 加入公开赞赏二维码和双语赞助页面。
- 将静态文档发布到 `/cpa_orbit/`，不影响服务器原有主页。

## v1.0.2

- 将订阅归档确立为唯一事实源，CPA 运行目录改为可重建投影。
- 加入单个与批量导入、内容去重、价格记录和自动额度检查。
- 加入 K12 / GPT Plus 双报价、14 天真实均价历史和提醒。
- 完成鲁班余额、号码、验证码轮询与释放流程。
- 增加深色模式、桌面宿主、托盘、通知与共享运行时。
- 建立开源贡献、安全、支持、文档和发布基础。
