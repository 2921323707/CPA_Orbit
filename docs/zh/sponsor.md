# 赞助项目

CPA Orbit 由个人独立维护。赞助用于维护、测试、文档和跨平台发布，不会购买功能优先级，也不会改变项目的隐私边界。

## 赞助支持什么

- 复现和修复真实使用问题。
- Windows 与 macOS 发布工作。
- 回归测试与安全检查。
- 中英文文档维护。
- 发布、签名与基础设施成本。
