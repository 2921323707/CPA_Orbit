# 架构决策

ADR 记录会长期影响数据所有权、信任边界、外部集成或部署方式的决定。

1. 订阅归档作为运行时事实源。
2. 双价格源与规范化内容身份。
3. 控制台分页、鲁班主题与双序列。
4. 鲁班短信会话代理。
5. 版本、语言与公开仓库边界。
6. 共享桌面运行时与独立健康信号。
7. 网关目标、托管归属与主备部署绑定。
8. Provider/日期订阅归档。
9. 安全 Auth JSON 预检与显式网关分配。
