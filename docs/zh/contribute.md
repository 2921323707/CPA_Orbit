# 参与贡献

CPA Orbit 欢迎边界明确、可以复现且不会泄露隐私的贡献。

## 可以从这里开始

- 提交可复现的错误报告。
- 用真实使用场景描述功能建议。
- 修正文档或补充故障排查证据。
- 修改持久化或信任边界前先阅读[系统架构](/zh/architecture/)。

## 提交前检查

- [ ] 改动只解决一个明确问题。
- [ ] 相关测试与构建通过。
- [ ] 用户行为与文档一致。
- [ ] 不包含真实 JSON、Token、密钥、账号列表、日志或私人截图。
- [ ] 持久化、信任边界和外部 API 变化已补充 ADR。
- [ ] Pull Request 说明问题、结果、风险和验证方式。

完整规则见[英文贡献规范](/CONTRIBUTING)，安全问题请遵循[私密报告流程](/zh/SECURITY)。
