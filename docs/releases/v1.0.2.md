# CPA Orbit v1.0.2

Released / 发布：2026-07-18

## 中文摘要

这是 CPA Orbit 的第一版公开发布。它把订阅归档、CPA 运行池、K12 / GPT Plus 报价、额度检查和鲁班接码放进同一个本地工作台，也把今晚处理过的分页、深色模式、加载动效、项目标志和开源基础文件一起收好。没有追求花哨的“大平台”叙事，先把每天真正要用的路径做顺。

## English summary

This is CPA Orbit's first public release. It brings subscription archives, the CPA runtime pool, K12 / GPT Plus pricing, quota checks, and the Luban SMS flow into one local workspace, along with the pagination, dark mode, loading motion, project mark, bilingual shell, and open-source foundations built during the first pass.
