# CPA Orbit v1.1.0

Released: July 19, 2026

CPA Orbit v1.1.0 is the first complete Windows desktop release of the local-first operations console. It unifies the desktop application and browser console around one Go runtime, one mutable data directory, and one subscription source of truth.

## Highlights

- Added the Wails desktop application with a fixed 1280×800 workspace, a native system tray, close-to-tray behavior, startup-at-login, native notifications, and taskbar alerts.
- Added one-click startup for the embedded Monitor API and automatic discovery or reuse of a local CLIProxyAPI runtime.
- Separated Monitor API health from CLIProxyAPI health so an unavailable proxy can no longer mark the embedded backend offline.
- Unified desktop and browser settings, API keys, subscription archives, quota state, alerts, and price history.
- Added Auto, Light, and Dark appearance modes and removed resize and scroll-observer behavior that caused visible interface flicker.
- Added subscription asset insights for recorded cost, average acquisition price, account health, and seven-day expiry risk.
- Added categorized documentation, architecture diagrams, public showcase assets, GitHub community templates, CI, Dependabot, and Playwright end-to-end coverage.

## Import reliability

- Fixed the desktop import action when an acquisition price is supplied.
- Kept multipart uploads file-only and moved the optional price to a validated request parameter for WebView2 compatibility.
- Added bounded request timeouts and guaranteed action-state recovery so an import cannot remain stuck on `Importing` after a client-side or network failure.
- Removed the blocking browser confirmation for an empty optional acquisition price.

## Windows download

Download `CPA-Orbit-v1.1.0-windows-amd64.exe` from the GitHub Release assets. The executable is a portable Windows x64 build and requires Microsoft Edge WebView2, which is included with supported Windows 10 and Windows 11 installations.

This community build is not Authenticode-signed, so Windows SmartScreen may display an unrecognized-publisher warning. Verify the SHA-256 checksum before running the executable.

CLIProxyAPI is intentionally not embedded because it contains an independent runtime and local configuration. CPA Orbit will reuse a service already listening on `127.0.0.1:8317`, discover a `cpa/cli-proxy-api.exe` or `cpa/app/cli-proxy-api.exe` directory beside the application, or use the `CPA_ORBIT_CPA_EXECUTABLE` and `CPA_ORBIT_CPA_CONFIG` environment variables.

Mutable application data is stored under `%AppData%\CPA Orbit` for a standalone copy. Imported credentials, tokens, local settings, and runtime logs are not included in the release executable.

## Integrity

The release also includes `CPA-Orbit-v1.1.0-SHA256SUMS.txt`. Verify the executable before distribution:

```powershell
Get-FileHash .\CPA-Orbit-v1.1.0-windows-amd64.exe -Algorithm SHA256
```
