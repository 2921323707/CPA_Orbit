# Support

- Use the in-app `/docs` page for setup and troubleshooting.
- Use a bug report for reproducible CPA Orbit defects.
- Use a feature request for product proposals.
- Do not post OAuth JSON, API keys, account email lists, full logs, or screenshots containing private data.
- Third-party payment, PriceAI, Luban, and CLIProxyAPI account issues belong to the corresponding service provider.

The public support URL will be added when the GitHub repository address is configured.
